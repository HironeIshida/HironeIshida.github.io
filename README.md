# HironeIshida.github.io

Static site for GitHub Pages.

Directory structure:

- `index.html`: redirects to the Japanese top page
- `ja/`: Japanese pages
- `en/`: English pages for research-related content (`Home`, `CV`, `Outputs`)
- `assets/css/`: shared stylesheets
- `assets/docs/`: shared PDF documents
- `assets/js/`: shared scripts
- `assets/images/`: shared images and icons
